EduConnect Hub - Website Plan

Site Name

EduConnect Hub

This name reflects a platform that connects students with educational resources and opportunities.

Logo: A simple book icon representing knowledge and learning.

Site Purpose

The website provides students with access to learning materials, academic guidance, and educational opportunities. It also helps users discover scholarships and study tips.

Scenarios

  • Where can I find study materials for my exams?
  • How can I apply for scholarships or educational programs?

Color Schema

Primary Color: #2E86AB – Used for headings and navigation

Secondary Color: #F6F5AE – Used for background

Typography

Font 1: Arial – Used for body text

Font 2: Georgia – Used for headings
